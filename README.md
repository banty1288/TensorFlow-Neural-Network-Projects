# 🚀 Projects Included

## 1️⃣ Breast Cancer Classification using ANN

### Objective
Predict whether a tumor is Benign or Malignant using a Deep Neural Network.

### Dataset
Breast Cancer Wisconsin Dataset (Scikit-Learn)

### Techniques Used

- Train-Test Split
- StandardScaler
- Artificial Neural Network (ANN)
- Binary Cross Entropy Loss
- Adam Optimizer
- Sigmoid Activation

### Model Architecture

Input Layer (30 Features)
↓
Dense (16, ReLU)
↓
Dense (8, ReLU)
↓
Dense (1, Sigmoid)

### Results

- Test Accuracy: ~96%
- Test Loss: ~0.13

### Concepts Learned

- Binary Classification
- Sigmoid Activation
- Binary Cross Entropy
- ANN Fundamentals
